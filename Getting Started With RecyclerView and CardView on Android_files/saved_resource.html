                For Click Listener With Position Do the Following things:
                1. Goto RVAdapter.java
                Replace:

                List<person> persons;
                RVAdapter(List<person> persons){
                this.persons=persons;
                }

                With

                List<person> persons;
                Context context;
                RVAdapter(List<person> persons,Context context){
                this.persons=persons;
                this.context=context;
                }

                Now, Go Little down you will find onBindViewHolder override Method.
                Goto Last Line and add

                holder.personphoto.setOnClickListener(new View.OnClickListener() {
                @Override
                public void onClick(View v) {
                Toast.makeText(context, "Index position is: "+position, Toast.LENGTH_SHORT).show();
                }
                });

                Above, is the click listener for the photo. Don't forget to change it, according to your needs. Hope it help *_* .
                Vote up so that every visitors can see it. Thank you..

                Oh.. I will tell you one more Plus point
                For GridLayoutManager
                Goto RecyclerViewActivity.java
                Remove LinearLayoutManager llm=new LinearLayoutManager(this);
                and add
                GridLayoutManager glm=new GridLayoutManager(this,4);
                here, the first parameter this is the context of activity and the 4 represents that the grid will contain 4 data in rows. :D
                Cheers!!
